## 🧠 Sobre o projeto

O **Task Flow** é uma aplicação web de **gestão de tarefas**, desenvolvida com foco em **produtividade**, **UX/UI** e **persistência de dados**.

A ideia não foi apenas criar uma To-Do List, mas sim estruturar um sistema real, com **autenticação de usuários**, **dados por perfil** e uma experiência fluida.

---

## ✨ Funcionalidades

- 🔐 Autenticação de usuários  
- 📝 CRUD completo de tarefas  
- 🔍 Filtros dinâmicos (status/categoria)  
- 💾 Persistência de dados por usuário  
- 📱 Interface responsiva e moderna  

---

## 🛠️ Tecnologias

- **React + TypeScript (Vite)**  
- **Tailwind CSS**  
- **Supabase (Auth + Database)**  
- **Vercel (Deploy / CI-CD)**  

---

## 🧩 Arquitetura

O projeto foi estruturado em três pilares:

- **Interface (UI)** → Experiência do usuário  
- **Lógica (Estado)** → Gerenciamento de dados (CRUD)  
- **Dados (Backend)** → Autenticação e persistência  

---

## 📈 Aprendizados

- Organização de projetos frontend com **componentização**  
- Controle de fluxo com **State Management**  
- Integração real com **backend e banco de dados**  
- Aplicação de boas práticas de **Clean Code**  
- Evolução de **design com base em estrutura sólida**  

---

## 🎯 Objetivo

Criar uma aplicação simples na interface, mas com uma base sólida de **frontend + backend**, pronta para evoluir e escalar.
